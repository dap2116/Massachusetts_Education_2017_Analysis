Massachusetts Education 2017 Analysis

This project analyzes Massachusetts public school data (2017) using Tableau to identify trends and performance gaps in graduation rates, college attendance, and 4th Grade MCAS Math results.

Key Analyses

Low Graduation Rates: Bar chart highlighting schools with the lowest graduation percentages.

College Attendance vs Class Size: Scatter plot showing the relationship between class size, college attendance, and economic disadvantage.

4th Grade MCAS Math: Interactive line chart comparing district-level math proficiency against a benchmark.

Tools & Skills

Tableau • Data Visualization • Educational Data Analysis • Performance Benchmarking

Conclusion

The analysis reveals significant disparities across schools and districts, supporting data-driven insights for educational improvement.
